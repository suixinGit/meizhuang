DualStyleGAN
DualStyleGAN通过渐进的迁移学习方法在小规模数据上学习外部风格，能够在保持自身人脸领域风格建模的同时，接受外部动漫人脸风格上的条件控制，有效地模仿动漫的画风，实现高清的动漫人脸风格化。此外，DualStyleGAN 对训练数据非常高效，只需大约 200 张图片，便可以训练得到不错的结果。
https://github.com/williamyang1991/DualStyleGAN
